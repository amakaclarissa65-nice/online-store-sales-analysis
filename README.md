# Online Store Sales Analysis

## Project Overview
This project analyzes an online store dataset to understand sales performance, customer behavior, and product trends. The goal is to generate actionable insights that can support business and inventory decisions.

## Tools Used
- SQL
- Excel

## Business Questions Answered
- Which products generate the highest revenue?
- Which customers contribute most to total sales?
- What sales trends exist over time?
- Which product categories perform best and worst?

## Analysis Performed
- Cleaned and validated transactional sales data
- Used SQL joins, subqueries, and aggregations to analyze sales and customer data
- Calculated revenue, order counts, and average sales metrics
- Identified top-performing products and categories
- Analyzed customer purchase patterns

## Key Insights
- A small group of products contributed a large portion of total revenue
- Certain categories consistently outperformed others in sales
- Repeat customers showed higher average purchase values
- Sales trends revealed peak purchasing periods

## Deliverables
- SQL queries for analysis
- Excel summary reports for revenue and performance
